# 斗罗大陆魂师对决BT服网站

## 📁 文件说明

### 核心文件
- `index.html` - 主页面HTML结构
- `styles.css` - 样式文件
- `script.js` - JavaScript交互功能

### 图片资源
- `logo.svg` - 网站Logo
- `favicon.ico` - 网站图标
- `image.png` - 主要展示图片

### 游戏内容图片 (需要替换)
- `game1.png` - 精彩游戏场景
- `game2.png` - 斗罗大陆世界
- `game3.png` - 魂师对决时刻
- `game4.png` - 精彩战斗场面
- `game5.png` - 游戏特色玩法
- `game6.png` - 魂师成长历程
- `game7.png` - 游戏精彩瞬间

## 🚀 启动网站

```bash
# 使用Python启动本地服务器
python3 -m http.server 8000

# 或者使用其他服务器
php -S localhost:8000
npx http-server
```

然后在浏览器中访问：http://localhost:8000

## 🎮 网站功能

- ✅ 完整的斗罗大陆魂师对决BT服介绍
- ✅ 邀请码、注册地址、下载链接
- ✅ QQ群信息和兑换码
- ✅ 7张游戏内容图片展示
- ✅ 响应式设计，支持移动端
- ✅ 图片点击放大查看功能
- ✅ 主题切换（浅色/深色/高对比度）

## 📝 使用说明

1. 将 `game1.png` 到 `game7.png` 替换为实际的游戏截图
2. 根据需要修改HTML中的链接地址
3. 可以直接编辑 `index.html` 来更新内容

## 📱 移动端支持

网站完全支持移动端访问，所有功能在手机上都能正常使用。
